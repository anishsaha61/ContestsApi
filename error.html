CodeChef Contests API

CodeChef Contests API provides details about all upcoming, ongoing and past CodeChef contests.

API Details:
Method Route Description
GET /codechef A JSON file containing all details about upcoming, ongoing and past codechef contests.
GET /codechef/future A JSON file containing all details about upcoming codechef contests.
GET /codechef/live A JSON file containing all details about ongoing codechef contests.
GET /codechef/past A JSON file containing all details about past codechef contests.
